\# SQL Student Data Analysis Project



\## Project Overview

This project focuses on analyzing student and course data using SQL. 

It demonstrates the use of SQL queries to extract meaningful insights from a relational database.



\## Database Structure

\- Students

\- Courses

\- Enrollments



\## Key SQL Concepts Used

\- SELECT

\- WHERE

\- GROUP BY

\- HAVING

\- JOIN

\- Aggregate Functions



\## Insights Generated

\- Total number of students

\- Average marks per course

\- High-performing students

\- Course-wise performance analysis



\## Tools Used

\- MySQL / SQLite

\- SQL queries
